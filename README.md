# october_test
L’idée globale : créer une API en Node.js, avec une seule route qui prend en paramètre le nom d’une entreprise française, et qui renvoie en JSON son numéro de téléphone, trouvé sur le web par les moyens de ton choix.


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
